# CUERRE PROJECT

CUERRE is a small system to create, share and managment QR codes for all you need.

## Getting Started

This system has developed with Spring Boot, Java and Maven.

### Prerequisites

What you need?

Maven 3.0 +
Java 8 +

### Installing

Clone this repository.
Go to main folder.

Run mvn spring-boot:run
